# Deploy to Heroku
## Set up
Download [Heroku Toolbelt](https://toolbelt.heroku.com/)
Enter in CLI:
```
heroku login
heroku git:remote -a lomaieva-medical
```
Will ask you for your heroku credentials.

## Deploy
```
git push heroku master
```
